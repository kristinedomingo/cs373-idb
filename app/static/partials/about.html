Data Sources
Spotify Web API
This website's main data source. It provided information on three main models (albums, artists, tracks) as well as embedded widgets that allow the user to play selected tracks.
Echo Nest API
A secondary data source that provided a lot of information about artists, including their biographies and recent news articles concerning them.
